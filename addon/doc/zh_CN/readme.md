# 蓝牙电量播报

`BlueToothBatteryReport` 用于播报本机蓝牙设备电量。

支持多设备电量播报，会顺序播报能检测到电量的所有蓝牙设备。

播报顺序为：已连接、未知状态、已断开。

## 热键

- `NVDA+Control+Shift+B`

## 播报格式

检测到电量信息时，会按如下格式播报：

- `设备名 电量 xx%`
